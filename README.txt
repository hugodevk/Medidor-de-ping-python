# Medidor de ping de python

Este script mide el tiempo del ping de un dispositivo especificado por el usuario

Uso: Pones la ip de tu dispositivo en este lugar

host = "192.169.1.91"   


Argumentos: la direccion ip del dispositivo que se quiere medir el ping.

Salida: Imprime el tiempo en milisegundos
